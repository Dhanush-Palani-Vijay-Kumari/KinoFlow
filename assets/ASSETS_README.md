# Assets

Place your font files here before running the app:

## Required fonts (download from Google Fonts)
- `assets/fonts/Poppins-Regular.ttf`
- `assets/fonts/Poppins-Medium.ttf`
- `assets/fonts/Poppins-SemiBold.ttf`
- `assets/fonts/Poppins-Bold.ttf`

Download from: https://fonts.google.com/specimen/Poppins

## Images
Place any local image assets in `assets/images/`

## Icons
Place any custom icon assets in `assets/icons/`

> Note: The app uses network images (Unsplash) for movie posters and backdrops,
> so no local movie images are required. Fonts are the only required local assets.
